# 低保真线框草图

## 设计方向

- 风格：治愈系、自然手账感、轻游戏化
- 布局策略：以“动物”为视觉中心，学习和消费操作围绕主视觉展开
- 目标：页面一眼看懂，核心按钮始终明确

## 页面布局原则

- 顶部保留返回与关键状态
- 中间聚焦主要行为对象
- 底部或中下区放主操作按钮
- 每页只保留一个最重要的 CTA
